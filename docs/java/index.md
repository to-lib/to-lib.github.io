# Java 编程

欢迎来到 Java 编程完整学习指南！本指南涵盖了 Java 编程语言的核心知识和实践技巧。

## 📚 学习内容

### 基础知识

- **基础语法** - 数据类型、变量、运算符、流程控制
- **面向对象** - 类、对象、封装、继承、多态
- **异常处理** - try-catch、自定义异常、最佳实践

### 核心特性

- **集合框架** - List、Set、Map 及其实现类
- **泛型编程** - 泛型类、泛型方法、通配符
- **注解** - 内置注解、自定义注解、注解处理器
- **反射机制** - Class 类、动态代理、元数据

### 高级主题

- **多线程** - Thread、线程同步、线程池
- **IO 流** - 字节流、字符流、NIO
- **函数式编程** - Lambda 表达式、Stream API
- **JVM 原理** - 内存模型、垃圾回收、性能调优

## 🚀 快速开始

如果你是 Java 初学者，建议按以下顺序学习：

1. [基础语法](./basic-syntax) - 掌握 Java 基本语法
2. [面向对象](./oop) - 理解面向对象编程思想
3. [集合框架](./collections) - 学习常用数据结构
4. [异常处理](./exception-handling) - 掌握异常处理机制
5. [多线程](./multithreading) - 了解并发编程基础

## 📖 学习路径

### 初级开发者

- 基础语法和数据类型
- 面向对象基础
- 常用集合类
- 异常处理基础

### 中级开发者

- 深入面向对象（接口、抽象类）
- 集合框架深入
- 多线程基础
- IO 流操作
- 泛型编程

### 高级开发者

- JVM 内存模型
- 并发编程深入
- 函数式编程
- 反射和动态代理
- 性能优化
