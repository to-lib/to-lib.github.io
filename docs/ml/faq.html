机器学习常见问题

基础问题​

机器学习、深度学习、AI 有什么区别？​

  • AI (人工智能)：让机器表现出智能行为的广泛领域
  • 机器学习：AI 的子集，从数据中学习模式
  • 深度学习：机器学习的子集，使用多层神经网络

监督学习和无监督学习的区别？​

类型数据目标示例
监督学习有标签预测标签分类、回归
无监督学习无标签发现结构聚类、降维

如何判断过拟合和欠拟合？​

  • 过拟合：训练准确率高，验证准确率低 → 正则化、更多数据
  • 欠拟合：训练和验证准确率都低 → 增加复杂度、更多特征

实践问题​

特征需要标准化吗？​

算法需要标准化原因
线性回归/逻辑回归是梯度下降收敛
SVM是距离计算
树模型否基于分裂
神经网络是训练稳定性

如何处理类别不平衡？​

  1. 类别权重：class_weight='balanced'
  2. 过采样：SMOTE
  3. 欠采样：RandomUnderSampler
  4. 调整阈值：根据 PR 曲线选择

如何选择模型？​

  1. 从简单模型开始（逻辑回归作为基线）
  2. 尝试随机森林/XGBoost
  3. 使用交叉验证比较
  4. 选择性能和复杂度的平衡

深度学习问题​

如何选择激活函数？​

位置推荐
隐藏层ReLU / LeakyReLU
二分类输出Sigmoid
多分类输出Softmax
回归输出无（线性）

学习率如何设置？​

  • 太大：震荡不收敛
  • 太小：收敛太慢
  • 推荐：从 0.001 开始，使用学习率调度器

什么时候使用预训练模型？​

  • 数据量小时
  • 领域相似时（如 ImageNet 预训练用于其他图像任务）
  • 通过微调适应新任务
