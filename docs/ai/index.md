# AI 开发概览

欢迎来到 AI 开发文档。这里汇集了关于人工智能应用开发的核心概念和技术指南，特别是针对大语言模型（LLM）的应用开发。

## 文档内容

### [🤖 AI Agent (智能体)](./agent)

深入了解 AI Agent 的概念、架构和应用场景。学习如何构建能够感知环境、通过决策采取行动的智能体。

### [🔌 MCP (模型上下文协议)](./mcp)

探索 Model Context Protocol (MCP)，这是一种标准化的协议，旨在将 AI 模型连接到各种数据源和工具，打破系统的孤岛。

### [📚 RAG (检索增强生成)](./rag)

学习 RAG 技术，了解如何通过检索外部知识库来增强 LLM 的生成能力，解决幻觉问题并提供最新的信息。

## 学习路线

建议按照以下顺序进行学习：

1.  **基础概念**：先了解 AI Agent 的基本工作原理。
2.  **数据连接**：学习 RAG 技术，解决数据供给问题。
3.  **系统集成**：掌握 MCP 协议，实现标准化集成。
