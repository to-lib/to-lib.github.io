# 快速参考

## 📊 数据结构复杂度

| 数据结构 | 访问     | 查找     | 插入     | 删除     |
| -------- | -------- | -------- | -------- | -------- |
| 数组     | O(1)     | O(n)     | O(n)     | O(n)     |
| 链表     | O(n)     | O(n)     | O(1)     | O(1)     |
| 栈/队列  | O(n)     | O(n)     | O(1)     | O(1)     |
| 哈希表   | -        | O(1)     | O(1)     | O(1)     |
| BST      | O(log n) | O(log n) | O(log n) | O(log n) |
| 堆       | -        | -        | O(log n) | O(log n) |

## 📊 排序算法复杂度

| 算法 | 平均       | 最坏       | 空间     | 稳定 |
| ---- | ---------- | ---------- | -------- | ---- |
| 冒泡 | O(n²)      | O(n²)      | O(1)     | ✅   |
| 选择 | O(n²)      | O(n²)      | O(1)     | ❌   |
| 插入 | O(n²)      | O(n²)      | O(1)     | ✅   |
| 归并 | O(n log n) | O(n log n) | O(n)     | ✅   |
| 快速 | O(n log n) | O(n²)      | O(log n) | ❌   |
| 堆排 | O(n log n) | O(n log n) | O(1)     | ❌   |

## 🎯 算法模式选择

| 问题类型     | 推荐算法      |
| ------------ | ------------- |
| 有序数组双向扫描 | 双指针     |
| 连续区间最值/计数 | 滑动窗口   |
| 区间统计/子数组计数 | 前缀和（+哈希） |
| 多次区间更新 | 差分         |
| 状态压缩/集合枚举 | 位运算/位掩码 |
| 有序数组查找 | 二分查找      |
| 最优解问题   | 动态规划/贪心 |
| 所有解问题   | 回溯          |
| 最短路径     | BFS/Dijkstra  |
| 连通性       | DFS/并查集    |
| 前 K 问题    | 堆/快速选择   |
