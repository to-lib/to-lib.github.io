# 数据结构与算法

欢迎来到数据结构与算法完整学习指南！本指南涵盖了计算机科学的核心基础知识和实践技巧。

> [!IMPORTANT]
> 本教程基于 **JDK 1.8 (Java 8)** 版本，代码示例均可在 Java 8 环境下运行。建议先完成 [Java 开发环境搭建](/docs/java/environment-setup)。

## 📚 学习内容

### 数据结构

- **线性结构** - 数组、链表、栈、队列
- **树形结构** - 二叉树、AVL 树、红黑树、B 树、堆
- **散列结构** - 哈希表、布隆过滤器
- **图结构** - 图的表示、遍历、最短路径

### 算法基础

- **排序算法** - 冒泡、选择、插入、归并、快速、堆排序
- **查找算法** - 顺序查找、二分查找、哈希查找
- **递归与分治** - 递归思想、分治策略、经典应用
- **动态规划** - 状态转移、记忆化搜索、常见问题

### 高级算法

- **贪心算法** - 贪心策略、最优子结构
- **回溯算法** - 回溯思想、剪枝优化
- **图算法** - DFS、BFS、Dijkstra、拓扑排序
- **字符串算法** - KMP、Rabin-Karp、Trie 树
