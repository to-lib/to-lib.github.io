# 常见问题

## 🤔 入门问题

### 如何选择合适的数据结构？

- 频繁随机访问 → **数组**
- 频繁插入删除 → **链表**
- 后进先出 → **栈**
- 先进先出 → **队列**
- 快速查找 → **哈希表**
- 维护顺序 → **树**

### 如何分析算法复杂度？

1. 计算循环执行次数
2. 忽略常数因子
3. 保留最高阶项

## 🎯 算法问题

### 递归 vs 迭代？

- 递归代码简洁，但有栈溢出风险
- 迭代效率更高，但代码可能复杂
- 可以用栈模拟递归

### 动态规划 vs 贪心？

| 特点       | 动态规划 | 贪心     |
| ---------- | -------- | -------- |
| 子问题重叠 | 需要     | 不需要   |
| 最优保证   | 总是     | 有时     |
| 复杂度     | 通常更高 | 通常更低 |

### 什么时候用回溯？

- 需要找出所有解
- 问题有约束条件
- 可以通过剪枝优化

## 💡 刷题建议

1. 先理解题意和示例
2. 分析输入规模确定复杂度
3. 选择合适的数据结构
4. 写出伪代码再编码
5. 注意边界条件
